# Marvel Vault - Frontend Challenge

SPA com listagem e detalhe de personagens da Marvel.

## Tecnologias
- Next.js (App Router)
- TypeScript
- React
- ESLint + Prettier
- Zustand (estado global)
- Tailwind CSS

## Como rodar

```bash
npm install
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
